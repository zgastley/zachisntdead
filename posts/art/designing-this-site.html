Designing This Site

This site is intentionally spare. The work should feel present without the interface trying to compete with it. I wanted navigation to fade into the background, type to feel personal, and the layout to breathe. The tone is a studio log, not a portfolio splash.

Instead of a single headline hero, the site starts with a quiet motif and a simple statement. That “Not yet.” anchor sets a tone: the work is ongoing. It signals process over polish and gives the page a soft, honest pulse.

Palette

The palette is warm and grounded — parchment, deep ink, and a few accents pulled from the head artwork. Color stays restrained so embeds, images, and writing can carry the weight. It also makes dark mode feel like a true alternate room instead of a filter slapped on top.

Layout + Rhythm

Spacing does most of the heavy lifting. Sections follow a steady rhythm: title, short summary, then content. The occasional soft card adds contrast without breaking alignment. I like the feeling of a long page you can exhale into.

The live status bar is both playful and practical. It’s a tiny proof‑of‑life and a subtle narrative device: this page updates because the person behind it is still here, still making.

Dark Mode

Dark mode is tuned, not toggled. The panels get deeper, the text warms up, and the accent shifts so the work stays legible without feeling harsh.

Technical Notes

Technically, the site is a single static page hosted on GitHub Pages. The content is split into section files and stitched together with a tiny Python build script. That lets me keep one page in production while editing each section in isolation.

The live heart rate comes from a public Google Sheet updated by a Fitbit automation, fetched in the browser every 30 seconds. The gallery component is plain JavaScript — no frameworks, just a simple image swap and a small thumb grid.

Embedded media is treated like content, not a separate widget. Bandcamp and YouTube live inside the same rhythm so the page still reads as a single story.

In short: a site designed to hold work, not perform for it. Minimal, but not sterile. Structured, but still curious.
